# RastaFlow AI

## Intelligent Traffic Incident Prediction and Diversion Management System

RastaFlow AI is an AI-powered traffic management platform that helps traffic authorities assess road incidents and make informed decisions. The system predicts incident severity, estimates clearance time, visualizes congestion using heatmaps, and suggests diversion routes to improve traffic flow.

## Features

* Incident Severity Prediction
* Clearance Time Estimation
* Traffic Congestion Heatmaps
* Diversion Route Suggestions
* AI-Based Traffic Recommendations
* Interactive Dashboard

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Start the backend server.
2. Start the frontend application.
3. Open the application in the browser.
4. Enter incident details.
5. View predictions, heatmaps, diversion routes, and recommendations.
